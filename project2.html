Personal Identity

A website that highlights photographs.

As a photographer I wanted to design and build a simple yet informative website that showcased my work and highlighted the photographs above all. It needed to be modern, bold and playful so it could stand out among the different types of photography portfolios. Also, I really wanted to take care of every single detail in order to have an unique look.

On the one hand, the website was designed with the idea of popping out the photographs harmoniously. So I wanted to implement light boxes on a single page layout. Giving the website the ability to have a flexible yet simple setting. On the other hand, after some concepting, I was inspired by an illustration of a self-portrait I did previously on Adobe Illustrator, this particular illustaration has a very low poly style and plays a lot with color shades. So I wanted this website to have the same feeling by using a vibrant colour scheme and a bold typography.
